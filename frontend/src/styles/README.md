# Frontend

Frontend workspace for the CampusProject application.

## Suggested Commands

```bash
npm install
npm run dev
```

## Directory Guide

- `public/`: static public assets
- `src/assets/`: imported images, icons, and style assets
- `src/components/`: reusable UI components
- `src/pages/`: route-level pages
- `src/services/`: API client modules
- `src/router/`: route definitions
- `src/stores/`: shared frontend state
- `src/utils/`: frontend helpers
